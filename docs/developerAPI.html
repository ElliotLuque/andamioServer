Andamio API Specification - Developer

Endpoints

Entity Operation URL Method Profile Description
developer get localhost:8082/developer/2 GET developer Get developer info, (can only see his info)
developer get localhost:8082/developer/1 GET admin Get every developer info
developer get localhost:8082/developer/3 GET reviewer Get developer info, (can only see his students info)
developer count localhost:8082/developer/count GET admin Gets count of all developers
developer getPage localhost:8082/developer/ GET admin Get a pagination of developer's info
developer create localhost:8082/developer/ POST admin Create a new developer (assign a team on creation)
developer create localhost:8082/developer/ POST reviewer Create a new developer (assign a team on which the teacher is the developer)
developer update localhost:8082/developer/4 PUT admin Update a developer
developer update localhost:8082/developer/4 PUT reviewer Update a developer (can only update his students info)
developer delete localhost:8082/developer/4 DELETE admin Delete a developer
developer delete localhost:8082/developer/4 DELETE reviewer Delete a developer (can only delete his students)
