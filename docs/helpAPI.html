WildCart API Specification

Endpoints

Entity Operation URL Method Profile Description
help get localhost:8082/help/{id} GET developer Get the help of a user in session,he can only receive those he create
help get localhost:8082/help/{id} GET reviewer Get the help of a user in session,the teacher can consult the help of the members of his team
help get localhost:8082/help/{id} GET admin Get the help of a user in session,the admin can receive all existing helps
help create localhost:8082/help/ POST developer Can create new help adding only developers who are in his team, and resolutions which he works on.
help create localhost:8082/help/ POST reviewer Can create new help adding only developers who are in his team.
help create localhost:8082/help/ POST admin Can create new help adding only developers who are in the same project.
help update localhost:8082/help/ PUT developer Can update only help which he creates.
help update localhost:8082/help/ PUT reviewer Can update only helps of his team.
help update localhost:8082/help/ PUT admin Can update any help on project.
help delete localhost:8082/help/{id} DELETE developer Can delete only helps which he creates.
help delete localhost:8082/help/{id} DELETE reviewer Can update helps which he creates and created helps by his team.
help delete localhost:8082/help/{id} DELETE admin Can update any help on the project
help count localhost:8082/help/count GET developer Can get only the total of helps which were created by him.
help count localhost:8082/help/count GET reviewer Can get the total of helps that he creates and which were create by his team on the same project
help count localhost:8082/help/count GET admin Can get the total of helps in any project.
help participating localhost:8082/help/{id} GET developer Can consult the helps in which appears his id.
help participating localhost:8082/help/{id} GET reviewer Can consult the helps in which appears the developer with the specified id.Only can consult the helps of his team.
help participating localhost:8082/help/{id} GET admin Can consult any help in which appears the developer with the spicified id.
