team API Specification

Entity Operation URL Method Profile Description
team get localhost:8082/team/5 GET admin Returns team ID, team name and amount of developers
team get localhost:8082/team/5 GET reviewer Returns team ID, team name and amount of developers if reviewer owns said team
team get localhost:8082/team/5 GET user Access denied
team post localhost:8082/team/ POST admin Creates a team using info from request body (team name and creator's ID)
team post localhost:8082/team/ POST reviewer Creates a team using info from request body (team name and creator's ID)
team post localhost:8082/team/ POST user Access denied
team put localhost:8082/team/ PUT admin Updates team's name. Request body contains team's ID and updated name
team put localhost:8082/team/ PUT reviewer Updates team's name if team is owned by reviewer. Request body contains team's ID and updated name
team put localhost:8082/team/ PUT user Access denied
team delete localhost:8082/team/5 DELETE admin Deletes team by nullifying id_team from develpers that are in said team
team delete localhost:8082/team/5 DELETE reviewer Deletes team if owned. Nullifies id_team from develpers that are in said team
team delete localhost:8082/team/5 DELETE user Access denied
